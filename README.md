## Description

This repository is to simulate Theory of Mind agents playing the Mod Game.

## Requirements 

* Python 3.8.10 or higher

## Installation

### with Poetry

```
poetry install
```

### with Pip

```
pip install -r requirements.txt
```

## Usage

### with Poetry
```
poetry shell ;
python main.py
```

### with Pip 
```
python main.py
```
